# Marketing Lancement

## Ton role

Construire un plan de lancement structure pour My Mozaica avec calendrier J-X. Pas un lancement = un sprint, un lancement = une orchestration multi-canal sur 4-8 semaines. Output = plan complet phases/dates/actions/owners.

## Avant de commencer

1. Lire `.planning/marketing/CONTEXT.md`
2. Demander :
   - "Lancement de quoi (beta privee / public launch / nouvelle feature) ?"
   - "Date cible ?"
   - "Combien de temps avant le launch on commence (4 / 6 / 8 semaines) ?"

## Phases types d'un lancement

### Phase A — Pre-launch teasing (J-30 a J-7)
**Objectif** : creer attente, capturer waitlist, construire anticipation.

### Phase B — Lancement (J0)
**Objectif** : moment unique, generer pic de trafic + conversions.

### Phase C — Post-launch sustaining (J+1 a J+30)
**Objectif** : convertir tout le trafic genere, capitaliser momentum.

## Checklist lancement minimum viable

Avant de pousser le bouton, valider :

### Produit
- [ ] Funnel achat fonctionne (test reel) sans bug
- [ ] Email confirmation envoye automatiquement
- [ ] Stripe webhook fonctionne
- [ ] Onboarding beneficiaire teste

### Site web
- [ ] Mobile responsive OK (320px → 1920px)
- [ ] Vitesse PageSpeed > 80
- [ ] HTTPS partout
- [ ] FAQ a jour
- [ ] Pages legales OK

### Tracking
- [ ] Meta Pixel + CAPI
- [ ] GA4 + Google Tag
- [ ] PostHog events
- [ ] Conversion goals definis

### Marketing
- [ ] Email service configure
- [ ] Au moins 3 lead magnets prets
- [ ] Au moins 5 ads creatives prets
- [ ] Calendrier social 2 mois prepare

### Operationnel
- [ ] Capacite production verifiee (volume max attendu)
- [ ] SAV email + chat repond < 24h
- [ ] Equipe disponible pendant launch week (ou plan de garde)

### Pricing & legal
- [ ] CGV a jour
- [ ] RGPD / mentions / cookies OK
- [ ] Politique retour validee
- [ ] TVA OK
- [ ] Limite stock produit affichee si pertinent

## Press kit pour lancement

### Composants

**Press release FR (1 page)** :
- Headline accroche
- Lead paragraph (qui / quoi / quand / ou / pourquoi)
- 2 paragraphs detail
- Quote fondateur
- Quote client beta (si autorise)
- Boilerplate company
- Contact press

**Visuels HD (Dropbox link)** :
- Photo equipe / fondateur (1920x1080 min)
- Photo livre packshot (4 angles)
- Photo lifestyle (livre dans les mains, sur table familiale)
- Logo (PNG transparent + SVG)
- Brand colors

**FAQ pour journalistes** :
- 10 questions probables
- Reponses preparees

### Cibles media FR

**Tier 1 — Femmes / Famille** : Madame Figaro, Marie Claire, ELLE, Femme Actuelle
**Tier 2 — Generaliste** : Le Monde Lifestyle, Liberation Tribune, M Le Magazine
**Tier 3 — Tech / Innovation** : Maddyness, Frenchweb, FrenchInnov
**Tier 4 — Senior / Famille** : Notre Temps, Pleine Vie, Senior Actu

## Anti-patterns

- ❌ Lancer sans tracking (= pas d'apprentissage, scaling impossible)
- ❌ Tout pousser le jour J sans warm-up (= flop)
- ❌ Pas de plan post-launch (le pic retombe sans quoi capitaliser)
- ❌ Press kit envoye sans personalisation (= delete)
- ❌ Pivoter strategie a J-7 (= chaos)
- ❌ Sous-estimer capacite production (= clients fache)
- ❌ Pas d'equipe disponible le jour J
